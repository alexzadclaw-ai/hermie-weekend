# Event submission form fields

Use Tally, Google Forms, Airtable Forms, or Typeform free tier.

## Form title
Submit an event or local hidden gem

## Form description
Hermie Weekend shares short Friday picks for people near Naperville and the western suburbs. Submissions are reviewed for fit. Inclusion is not guaranteed.

## Fields
1. Event/place name — short text — required
2. Website or event link — URL — required
3. Date/time — short text — optional for always-available places
4. Location/address — short text — required
5. Best fit — multiple choice
   - Date night
   - Family/kids
   - Cheap/free
   - Rainy-day backup
   - Hidden gem
   - Food/drink
   - Live music/show
   - Class/workshop
6. Cost — short text
7. Why should locals care? — paragraph
8. Contact email — email — required
9. Is this a paid/sponsored placement request? — yes/no

## Confirmation message
Thanks — your submission was received. If it’s a good fit for an upcoming Friday list, Hermie Weekend may include it. Paid placements, if any, are clearly labeled.
