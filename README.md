# Launch checklist

## What the user must do manually
1. Create a public Telegram channel.
2. Choose a public handle.
3. Copy `copy/telegram-channel.md` description and pinned post.
4. Event submission form is set to: https://tally.so/r/b5209g
5. Telegram channel is set to: https://t.me/hermieweekend
6. Publish the site on GitHub Pages or Cloudflare Pages.

Published URL: https://www.hermieweekend.fun/
7. Generate a QR code pointing to the landing page.
8. Replace the QR placeholder in `flyers/qr-flyer.html` before printing.

## Suggested no-cost stack
- Telegram public channel: free
- Tally or Google Forms: free
- GitHub Pages or Cloudflare Pages: free
- QR code: browser/local generator, free
- Domain: optional later

## First 7 days
Day 1:
- Create channel and landing page.
- Post pinned intro.
- Add first soft-launch post.

Day 2:
- Create event submission form.
- Email 10 local venues using the event submission template.

Day 3:
- Print 10 QR flyers/cards.
- Ask 3 coffee shops/community boards if you can place one.

Day 4:
- Draft first real Friday post.
- Collect 15 candidate events/places.

Day 5:
- Publish first Friday roundup.
- Ask every early subscriber to forward it to one local friend.

Day 6–7:
- Review subscriber count and replies.
- Adjust categories based on what people engage with.

## Success threshold before spending money
- 50 subscribers: continue manually.
- 100 subscribers: buy a domain.
- 250 subscribers: test sponsor outreach.
- 500 subscribers: consider $50–$100/month local ads or paid placements.
